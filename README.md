# Templodocs

A sacred tool for documentation templates. So far you can use it to:
* generate word document from tempo csv files

## Usage

Generate a Word document from a template. To generate the word document run following 
command.

```bash
bin/templodocs path/to/data.csv path/to/template.docx output_filename
```
